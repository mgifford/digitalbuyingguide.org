Privasi

Panduan Pembelian Digital disusun dan dikelola oleh Layanan Digital Pemerintah (Government Digital Service atau GDS) Inggris. Data pribadi diproses sesuai dengan pernyataan privasi GOV.UK yang dapat dilihat di sini.

Data yang kami kumpulkan

Data pribadi yang kami kumpulkan dari Anda meliputi:

  • pertanyaan, permintaan informasi, atau umpan balik yang Anda berikan, termasuk alamat email Anda jika Anda menghubungi kami atau menanggapi survei
  • alamat Protokol Internet (Internet Protocol atau IP) Anda, dan perincian versi peramban web yang Anda gunakan
  • informasi tentang cara Anda menggunakan situs ini, menggunakan cookie dan teknik penandaan laman

Kami menggunakan perangkat lunak Google Analytics untuk mengumpulkan informasi tentang cara Anda menggunakan situs web kami. Ini termasuk alamat IP. Data ini dianonimkan sebelum digunakan untuk pemrosesan analisis.

Kami tidak menyimpan informasi pribadi Anda melalui Google Analytics (misalnya, nama atau alamat Anda).

Kami tidak akan mengidentifikasi Anda melalui informasi analisis, dan tidak akan menggabungkan informasi analisis dengan perangkat lain dengan cara yang dapat mengidentifikasi jati diri Anda.

Kami terus menguji dan memantau pengendalian perlindungan data kami untuk memastikan pengendalian tersebut perlindungan tersebut efektif dan mendeteksi kelemahan.

Alasan kami memerlukan data Anda

Kami mengumpulkan informasi melalui Google Analytics untuk memahami cara Anda menggunakan situs dan layanan digital pemerintah. Kami melakukan ini untuk membantu:

  • memastikan situs memenuhi kebutuhan penggunanya

Kami juga mengumpulkan data untuk:

  • menanggapi masukan yang Anda kirimkan, jika Anda meminta kami memberikan tanggapan
  • memantau penggunaan situs untuk mengidentifikasi ancaman keamanan

Tautan ke situs web lain

Situs web kami berisi tautan ke dan dari situs web lain. Kebijakan privasi ini hanya berlaku untuk situs web kami.

Apabila Anda mengunjungi situs web lain dari situs web ini, baca kebijakan privasi di situs web tersebut untuk mengetahui hal yang dilakukan situs web tersebut terhadap informasi Anda.

Apabila Anda mengunjungi situs web kami dari situs web lain, kami dapat menerima informasi pribadi dari situs web lain. Anda disarankan untuk membaca kebijakan privasi situs web asal untuk mendapatkan informasi lebih lanjut tentang hal ini.

Dasar hukum kami untuk memproses data Anda

Dasar hukum untuk memproses data pribadi tercakup dalam pemberitahuan privasi Pemerintah Inggris yang dapat ditemukan di sini.

Hal yang kami lakukan terhadap data Anda

Data yang kami kumpulkan dapat dibagikan dengan departemen pemerintah, lembaga, dan badan publik lainnya. Data tersebut juga dapat dibagikan dengan pemasok teknologi kami, misalnya penyedia hosting kami.

Kami tidak akan:

  • menjual atau menyewakan data Anda kepada pihak ketiga
  • membagikan data Anda dengan pihak ketiga untuk tujuan pemasaran
  • menggunakan data Anda dalam analisis

Kami akan membagikan data Anda jika diwajibkan untuk melakukannya oleh hukum - misalnya oleh putusan pengadilan atau untuk mencegah penipuan atau kejahatan lainnya.

Berapa lama kami menyimpan data Anda

Kami hanya akan menyimpan data pribadi Anda selama:

  • diperlukan untuk tujuan yang ditetapkan dalam dokumen ini
  • hukum mewajibkan kami untuk menyimpannya

Menjaga keamanan data Anda

Kami berkomitmen untuk melakukan upaya terbaik agar data Anda tetap aman. Kami telah menerapkan sistem dan proses untuk mencegah akses atau pengungkapan data Anda secara tidak sah - misalnya, kami melindungi data Anda menggunakan berbagai tingkat enkripsi.

Kami juga memastikan pihak ketiga yang bekerja sama dengan kami menyimpan semua data pribadi yang mereka proses atas nama kami.

Menghubungi kami atau mengajukan keluhan

Hubungi Tim Privasi Layanan Digital Pemerintah (Government Digital Service atau GDS) Inggris jika Anda:

  • memiliki pertanyaan tentang apa pun dalam pemberitahuan privasi ini
  • merasa bahwa data pribadi Anda telah disalahgunakan atau ditangani dengan tidak tepat

gds-privacy-office@digital.cabinet-office.gov.uk

Perubahan kebijakan ini

Setiap perubahan yang dibuat terhadap kebijakan privasi ini akan diposting di laman web ini. Silakan kembali lagi untuk mendapatkan informasi tentang pembaruan atau perubahan kebijakan privasi kami.
