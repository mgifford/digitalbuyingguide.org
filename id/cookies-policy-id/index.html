Cookie

Cookie adalah file kecil yang disimpan di ponsel, tablet, atau komputer Anda saat Anda mengunjungi situs web.

Kami menggunakan cookie agar Panduan Pembelian Digital berfungsi dan mengumpulkan informasi tentang cara Anda menggunakan situs web kami.

Cookie penting

Cookie penting menjaga keamanan informasi Anda saat Anda menggunakan situs web kami. Kami tidak perlu meminta izin untuk menggunakannya.

Nama Tujuan Kedaluwarsa
analisis Mempertahankan status untuk cookie analisis opsional 1 tahun
cookie_notice Menyimpan pengaturan persetujuan cookie Anda 1 tahun
csrftoken Membantu mencegah serangan Pemalsuan Permintaan Lintas Situs (Cross-Site Request Forgery atau CSRF) 1 tahun
django_language Mempertahankan pilihan bahasa Sesi
sessionid Mempertahankan id sesi 2 minggu

Cookie analisis (opsional)

Dengan izin Anda, kami menggunakan Google Analytics untuk mengumpulkan data tentang cara Anda menggunakan situs web kami. Informasi ini membantu kami untuk menyempurnakan situs web kami.

Google tidak diizinkan untuk menggunakan atau membagikan data analisis kami dengan siapa pun.

Google Analytics menyimpan informasi anonim tentang:

Nama Tujuan Kedaluwarsa
_ga Memeriksa jika Anda telah mengunjungi situs web kami sebelumnya. Ini membantu kami menghitung jumlah pengunjung situs kami. 2 tahun
_gid Memeriksa jika Anda telah mengunjungi situs web kami sebelumnya. Ini membantu kami menghitung jumlah pengunjung situs kami. 24 jam
