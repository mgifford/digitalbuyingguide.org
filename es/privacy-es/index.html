Privacidad

La Guía para compras digitales fue creada y es mantenida por el Servicio Digital Gubernamental del Reino Unido. Los datos personales se procesan de acuerdo con la Declaración de privacidad del Gobierno del Reino Unido, que se puede encontrar aquí.

Qué datos recopilamos
  • preguntas, consultas o comentarios que usted deje, incluida su dirección de correo electrónico si se comunica con nosotros o responde a una encuesta
  • su dirección del Protocolo de Internet (IP) y la información sobre qué navegador web utilizó
  • información sobre cómo utiliza el sitio, mediante cookies y técnicas de etiquetado de páginas

Utilizamos el software de Google Analytics para recopilar información sobre cómo utiliza nuestro sitio web. Esto incluye las direcciones IP. Los datos se anonimizan antes de ser utilizados para el procesamiento analítico.

No almacenamos su información personal a través de Google Analytics (por ejemplo, su nombre o dirección).

No lo identificaremos a través de información analítica y no combinaremos información analítica con otros conjuntos de datos de manera de poder identificar quién es usted.

Continuamente probamos y monitoreamos nuestros controles de protección de datos para asegurarnos de que sean efectivos y detectar cualquier debilidad.

Por qué necesitamos sus datos

Recopilamos información a través de Google Analytics para ver cómo utiliza los servicios digitales del sitio y del gobierno. Lo hacemos para ayudar a:

  • asegurarnos de que el sitio satisfaga las necesidades de sus usuarios

También recopilamos datos para:

  • responder a cualquier comentario que nos envíe, si nos lo hubiera pedido
  • supervisar el uso del sitio para identificar amenazas de seguridad

Enlaces a otros sitios web

Nuestros sitios web contienen enlaces hacia y desde otros sitios web. Esta política de privacidad solo se aplica a nuestro sitio web.

Si visita otro sitio web desde este, lea la política de privacidad en ese sitio web para averiguar qué hace con su información.

Si llega a nuestro sitio web desde otro sitio web, podemos recibir información personal desde el otro sitio web. Debe leer la política de privacidad del sitio web del que vino para obtener más información al respecto.

Nuestro fundamento jurídico para procesar sus datos

El fundamento jurídico para procesar datos personales está cubierto por el aviso de privacidad del Gobierno del Reino Unido, que se puede encontrar aquí.

Qué hacemos con sus datos

Los datos que recopilamos pueden compartirse con otros departamentos, dependencias y organismos públicos. También pueden compartirse con nuestros proveedores de tecnología, por ejemplo, nuestro proveedor de alojamiento.

No haremos lo siguiente:

  • vender o alquilar sus datos a terceros
  • compartir sus datos con terceros con fines de marketing
  • usar sus datos para realizar análisis

Compartiremos sus datos si la ley nos exige hacerlo, por ejemplo, por una orden judicial o para prevenir fraudes u otros delitos.

Durante cuánto tiempo conservamos sus datos

Solo conservaremos sus datos personales mientras que:

  • sea necesario para los fines establecidos en este documento
  • la ley nos lo exija

Mantenimiento seguro de sus datos

Estamos comprometidos a hacer todo lo que podamos para mantener sus datos seguros. Hemos establecido sistemas y procesos para evitar el acceso no autorizado o la divulgación de sus datos; por ejemplo, protegemos sus datos utilizando diferentes niveles de cifrado.

También nos aseguramos de que cualquier tercero con el que tratemos mantenga seguros todos los datos personales que procesa en nuestro nombre.

Comuníquese con nosotros o plantee una queja

Comuníquese con el Equipo de Privacidad del Servicio Digital del Gobierno del Reino Unido si:

  • tiene alguna pregunta sobre cualquier tema en este aviso de privacidad
  • cree que se ha hecho un uso indebido o un mal manejo de sus datos personales

gds-privacy-office@digital.cabinet-office.gov.uk

Modificaciones a esta política

Cualquier modificación que se realice a esta política de privacidad se publicará en esta página web. Vuelva a ingresar a la página para consultar actualizaciones o modificaciones a nuestra política de privacidad.
